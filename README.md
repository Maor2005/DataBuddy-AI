 DataBuddy AI 
 Empowering the Next Generation of Developers with Gemma 4

DataBuddy AI is a privacy-first, local-hosted mentorship platform designed to help students and developers master Data Engineering and Python coding standards. 

 Key Features
- Local Inference:** 100% private and offline using Ollama.
- Gemma 4 Powered:** Deep analysis of code logic, efficiency, and professional standards.
- Educational Focus:** Explains the "Why" behind every fix, not just the "How".

 Tech Stack
- Model: Google Gemma 4
- Local Engine: Ollama
- Frontend: Streamlit
- Language: Python

 How to Run Locally

 1. Prerequisites
Ensure you have Ollama installed and the Gemma 4 model pulled:
ollama run gemma4

 2. Setup
Install the dependencies:
pip install -r requirements.txt

 3. Launch
Run the Streamlit application:
streamlit run app.py
